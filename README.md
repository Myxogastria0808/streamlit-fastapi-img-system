# Streamlit と FastAPI を組み合わせて作成した何か

## 想定している OS の環境

#### Windows11

## Streamlit の実行

```batch
start-streamlit.cmd
```

#### Streamlit を用いて作成した Web App を起動した際にアクセスする URL

```
http://127.0.0.1:8080/
```

## FastAPI の実行

```batch
cd api
start-fastapi.cmd
```

#### FastAPI を用いて作成した Web API を起動した際にアクセスする URL (API ドキュメントにアクセスするときの URL)

```
http://127.0.0.1:8000/docs
```
